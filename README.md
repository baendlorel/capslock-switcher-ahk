# CapsLock Switcher - AutoHotkey 输入法切换工具

CapsLock Switcher 是一个轻量级的 AutoHotkey 工具，允许你使用 CapsLock 键在中英文输入法之间切换，类似于 macOS 上的行为。消除了使用 Shift 键切换输入模式的不便。

## 功能

- **CapsLock 切换**: 按下 CapsLock 键在中英文输入模式之间切换
- **Shift+CapsLock 同步**: 按下 Shift+CapsLock 检查并显示当前输入模式
- **视觉反馈**: 显示提示通知，指示当前语言模式
- **系统托盘集成**: 右键点击系统托盘图标进行快速控制
- **开机启动选项**: 可设置为系统启动时自动运行
- **暂停/恢复**: 通过托盘菜单临时禁用/启用脚本
- **跨应用兼容性**: 适用于大多数 Windows 应用程序和输入法

## 系统要求

- Windows 7/8/10/11 (x64 或 x86)
- 已安装中文输入法 (微软拼音、搜狗、QQ拼音等)

## 安装方法

1. 从 [发布页面](#) 下载最新版本 (或自己构建)
2. 运行 `capslock-switcher-vX.X.X.exe`

## 注意事项

1. 此脚本的作用是将`CapsLock`映射到`Ctrl + Space`，因此需要先在设置中将输入法切换快捷键改为`Ctrl + Space`。
2. build.js脚本会自行寻找可能的AutoHotKey编译器位置，如果找不到的话，请将完整路径写在ahk-path.txt文件中。

## 使用方法

1. **基本切换**: 按下 `CapsLock` 键在中英文输入模式之间切换
2. **检查当前模式**: 按下 `Shift + CapsLock` 显示当前输入模式而不切换
3. **托盘菜单**:
   - 右键点击托盘图标访问选项
   - "开机启动": 切换 Windows 登录时自动启动
   - "暂时关闭/开启": 临时禁用/启用脚本
   - "退出": 退出应用程序

## 配置

该工具开箱即用，使用默认设置。高级用户可以修改：

1. **提示显示时间**: 修改脚本中的 `TOAST_HOLD_MS` (默认: 640毫秒)
2. **提示淡出速度**: 修改 `TOAST_FADE_INTERVAL_MS` 和 `TOAST_FADE_STEP`
3. **提示外观**: 修改 `ShowToast` 函数中的颜色和字体

## 故障排除

1. **脚本不工作**
   - 确保已安装 AutoHotkey v2.0
   - 检查输入法是否正在运行
   - 尝试以管理员身份运行

2. **提示不显示**
   - 检查脚本是否被暂停 (右键点击托盘图标)
   - 某些全屏应用程序可能会阻止提示通知

3. **输入法未检测到**
   - 当前脚本支持中文语言布局 (zh-CN, zh-TW, zh-HK 等)
   - 其他语言输入可能需要修改代码
